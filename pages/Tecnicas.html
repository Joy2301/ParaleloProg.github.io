TECNICAS MAS EMPLEADAS EN PROGRAMACION PARALELA

Técnicas de Programación Paralela:


1. Hilos (Threads)

Los hilos permiten la ejecución concurrente de múltiples subprocesos dentro de un programa. Cada hilo puede ejecutar una parte diferente del código de forma independiente, aprovechando así la capacidad de procesamiento paralelo de los sistemas multi-hilo. Esto es útil para tareas que se pueden dividir en subprocesos independientes.

2. Procesos

Los procesos permiten la ejecución simultánea de múltiples programas o instancias del mismo programa. Cada proceso tiene su propio espacio de memoria y recursos, lo que permite una ejecución paralela de tareas separadas. Los procesos pueden comunicarse y sincronizarse entre sí utilizando mecanismos como tuberías, colas o memoria compartida.

3. Vectorización

La vectorización es una técnica que aprovecha las unidades de procesamiento vectorial de los sistemas para realizar operaciones en paralelo en conjuntos de datos. Consiste en reorganizar el código para que las operaciones se realicen en elementos de datos en lugar de en elementos individuales, lo que puede mejorar significativamente el rendimiento en aplicaciones que realizan cálculos repetitivos o de procesamiento intensivo.

4. Paralelismo de datos

El paralelismo de datos implica dividir un conjunto de datos en partes más pequeñas y procesar esas partes simultáneamente en múltiples unidades de procesamiento. Cada unidad de procesamiento realiza la misma operación en diferentes porciones de datos. Esta técnica es útil en aplicaciones que realizan operaciones repetitivas y en las que los datos se pueden procesar de forma independiente.

5. Paralelismo a nivel de tarea

El paralelismo a nivel de tarea consiste en dividir una tarea grande en tareas más pequeñas y ejecutar esas tareas simultáneamente. Cada tarea puede ser asignada a un hilo o proceso diferente y ejecutarse en paralelo. Esta técnica es útil cuando las tareas no están estrictamente acopladas y se pueden ejecutar independientemente.
